AI Chat to PDF Extension

Supports ChatGPT, Claude, Grok, and Gemini

After downloading, install the extension:

  1. Unzip the downloaded AI_chat_to_PDF.zip file to a folder.
  2. Open Browser and go to Browser extension page. EX : chrome://extensions//
  3. Enable Developer mode (top-right).
  4. Click Drag and Drop unzipped AI_chat_to_PDF folder.
  5. The extension will be added to Chrome.
